IfDef_4.0.g4

- zeroIfDefSet으로 묶어주는 parse tree 생성


"#if 0" Checker
불필요 지시자인 "#if 0"와 이에 대한 짝인 "#endif"를 찾아서 parse tree를 만들어준다.
 이 문법 파일로 생성되는 .java 파일들을 이용하여 한 디렉토리 내의 모든 .c .h 파일에 대하여 조사한 후, 각 파일들에 대하여 파일명/줄번호/크기 등을 csv 로 출력해보는 프로그램을 만들어 보았었다.
 하지만 parse tree 가 잘 나오는 대신에 실행시간이 매우 길어졌다는 단점이 있었다. 따라서 현재는 #if 0 와 #endif 의 set 까지 문법파일에서 구분하지 않고, 자바에서 stack을 사용하여 #if 0 와 그에 걸맞는 #endif 가 언제 등장하는 지 파악한 후에 해당 라인의 크기를 측정하는 방법을 시도하고 있다.
